Shell C

cmshell

Minimal shell for ricers. A lightweight, blazingly fast UNIX shell written in C with zero dependencies. Designed for users who want a clean, customizable terminal experience without the weight of bash or zsh.

  • <5KB binary, zero runtime dependencies
  • Customizable prompt with ANSI color support
  • Built-in tab completion & history
  • Perfect for ricing setups and minimal distros

WIP Firmware

inSeaBIOS

Lightweight open-source BIOS implementation for minimal x86 systems. A modern fork of SeaBIOS trimmed down to the essentials — faster boot times, smaller footprint, and cleaner codebase for embedded and retro computing.

  • Sub-100ms POST time on modern hardware
  • Stripped down to ~32KB flash image
  • Legacy BIOS & basic UEFI CSM support
  • Ideal for QEMU/KVM virtual machines

WIP Rust

proboot

Fast and secure bootloader for modern Linux distributions. Written in Rust with safety guarantees, proboot replaces GRUB with a sleek, minimal alternative featuring instant boot, themeable menu, and native LUKS2 decryption.

  • Written in Rust — memory safe by design
  • Native LUKS2 & TPM2 unlock support
  • Customizable boot menu with TUI themes
  • Drop-in GRUB replacement with migration tool

Why infastracture

Principles that matter

Every line of code follows our core philosophy — minimalism, performance, and transparency.

Zero Bloat

Every project is stripped to its absolute essentials. No unnecessary features, no hidden dependencies, no surprises.

Blazing Fast

Sub-millisecond shell startup, sub-100ms BIOS POST, instant bootloader handoff. We obsess over latency.

Fully Open Source

MIT/GPL licensed. No telemetry, no phone-home, no premium tiers. The source is the documentation.

Ricer-Friendly

Built by ricers, for ricers. Deep theming, custom colors, and visual tweaking out of the box.

Security First

Memory-safe languages where possible. proboot uses Rust, cmshell avoids dynamic allocation.

Well Documented

Comprehensive READMEs, inline comments, and wiki pages. If you can't figure it out in 5 minutes, it's a bug.
